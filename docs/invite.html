Invite Scripty

Permissions Required

  • Manage Webhooks: this is for a planned feature (sending the messages as the user)
  • Send Messages: hopefully obvious
  • Manage Messages: for embed menus, that way the bot can remove the user's reaction, as well as when doing setup, the bot can delete the setup messages.
  • Embed Links: the bot almost entirely communicates via embeds.
  • Attach Files: this is for a planned feature (showing the raw response)
  • Read Message History: this is not currently required but is asked for just in case a future feature requires it.
  • Add Reactions: for embed menus, that way the bot can add the initial reaction.
  • Use External Emojis: this is not currently required but is asked for just in case a future feature requires it.
  • Connect: this is required for the bot to join the voice chat.

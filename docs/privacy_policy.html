Privacy Policy

What data is collected?

Scripty only stores the bare minimum required to operate. This includes the following:

  • Guild IDs
  • Channel IDs
  • Per-guild configuration
  • User IDs
  • Anonymous, aggregated statistics about bot function, including:
    • Discord Gateway events: only the total bot-wide count of events is stored.
    • Total milliseconds of audio transcripted: this is aggregated and cannot ever be linked back to a specific guild.
    • Guild count
    • User count

For what purposes is this data stored?

This data is stored for various purposes, including:

  • Storing configuration settings persistently
  • Providing feedback on what areas need to be worked upon

How is this data used?

Any personally identifiable data is only used to provide core functions of the bot.

Anonymous, aggregated stats are used for various purposes, including, but not limited to, providing feedback on the growth of the bot.

Who is the data shared with?

Personally identifiable data is shared with no one except the core developers (currently only 0/0#0001).

Anonymous, aggregated stats are shared with everyone who wishes to view them.

Who can I contact with concerns?

You can contact the core developers via Discord (0/0#0001) or via email (scripty @ this domain).

How can I request deletion of my data?

You may contact the core developers via the mediums specified above and request we delete your data.
